1) How does javascript works?

Answer: JavaScript works in browser with V8 engine. Inside V8 engine, JavaScript source code parse it and make it like a Syntax Tree, then it interpreted the code. JavaScript compiles the code with JIT(Just in Time) compiler. With this it optimized the machine code to bytecode. And by JavaScript garbage collection all unnecessary codes are removed.

2) What is Event Loop?

Answer: JavaScript has two things, during runtime, one is stack and another one is Queue. Stack works like, last in first out. Queue works like, first in first out. Event Loop manage the code to run squentially.Heap is also present in Event Loop, where objects are passed to it unstructured ways. And in Event Loop all values from stack returns then returns the value of Queue.In this way Event Loop works.

3) Difference between LocalStorage and SessionStorage.
LocalStorage:

Data is stored permanently untill it is cleared manually or through a setting. To save an item in LocalStorage we use localStorage.setItems(). To retrieve the items from LocalStorage by using localStorage.getItems(). In a new tab we can able to retrieve the item from the localStorage.

SessionStorage:

Data is stored in period of time. To save an item in SessionStorage we use SessionStorage.setItems(). To retrieve the items from SessionStorage by using SessionStorage.getItems(). In a new tab we can't able to retrieve the item from the SessionStorage.
